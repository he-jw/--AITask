# 毕业推文

## 推文

- 搜索官方毕业推文格式作为参考
- 编写 Prompt（指示生成“推文和封面图提示词”）
- AI 生成结果

## 投票结果统计

- 使用 Cursor 编写脚本统计投票结果

### 流程

- **输入**：一份从 `lists.apache.org` 保存下来的投票线程 HTML。
- **切分邮件**：按每个 `<div class="email_wrapper" id="email_...">` 把 HTML 切成“一封封邮件”。
- **逐封解析**：
  - 解析 **作者 + 时间**（用于后面“最后一票”判定）
  - 解析 **永久链接**（点进去能对应到那封邮件）
  - 解析 **正文**（取 `<pre class="chatty_body">...</pre>`，把 `<br>` 变换行、去掉其他标签，并删除引用块 `blockquote` 防止算到被引用内容）
  - 在正文里逐行找投票：遇到第一条像 `+1/-1/0/abstain(弃权)` 的行就认为这封邮件投了这一票
- **汇总口径**：按“作者”分组，同一作者多次投票只取时间上最后一封邮件的票作为最终票。
- **输出**：统计最终票的 `+1/-1/0/abstain` 人数（其中展示时把 `0 + abstain` 合并成“0（含 abstain）”），并列出每个人的链接/时间/摘录；如果有人改票，额外列出其历史投票记录。
